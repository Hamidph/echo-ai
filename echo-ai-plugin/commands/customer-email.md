### Email types:
- **Day 0 onboarding**: show first value in 5 mins — link to create first experiment
- **Day 3 activation** (no experiment run): show example result, remove friction
- **Day 14 re-engagement** (inactive): show what they're missing, data-led
- **Feature announcement**: what changed + why it matters to them specifically
- **Upgrade nudge**: show ROI, not features — "you're running out of prompts"
- **Support response**: acknowledge → solution → follow-up offer
- **Billing failure**: clear, non-scary, one-click fix link
